# 16-24个月每周培养计划

目标：从16个月开始，到24个月前后，以健康和运动基础为优先，围绕语言表达、认知探索、大运动、精细动作、社交情绪、饮食睡眠、视力保护、奶瓶和屏幕管理，持续培养一个聪明、健康、身体协调、情绪稳定、愿意沟通的宝宝。

## 怎么使用

1. 每周打开对应文档，只抓本周主题，不同时推进太多目标。
2. 每天做2-3个短场景，其中至少一个是亲子运动活动，比一次做很久更有效。
3. 所有表达都算：看、指、伸手、递物、发声、手势、词语。
4. 不用考试宝宝，不逼重复，不用屏幕哄饭、哄睡、止哭。
5. 如果宝宝状态不好，保留陪伴、睡眠、饮食和安全，训练可以暂停。

## 运动优先说明

每周计划都增加了“科学运动重点”和“周一到周日亲子活动”。运动不是单独训练，而是和语言、音乐、认知、生活自理结合：走、停、蹲、转身、推拉、搬运、球类、跨越、台阶准备、节奏控制。每天重复一个喜欢的亲子活动完全可以，重复能帮助宝宝建立身体记忆和安全感。

## 月度主线

| 月龄 | 主线 | 重点 |
| --- | --- | --- |
| 16个月 | 从“听得懂”走向“主动表达”，同时让走路、蹲起、杯饮和共同注意更稳定。 | 等待3秒和需求表达；稳定走路、蹲起、滚球；杯子喝水和睡前奶前移；回应名字、指物、模仿观察 |
| 17个月 | 扩大日常词和模仿动作，把亲子游戏变成轮流、请求、帮忙和整理。 | 高频词稳定；轮流游戏；精细动作和假装游戏；白天奶瓶继续减少 |
| 18个月 | 完成一次阶段复盘，重点看有意义表达、共同注意、模仿、走路和睡眠饮食。 | 18个月里程碑观察；两步指令萌芽；踢球和搬运；情绪命名 |
| 19个月 | 鼓励她用动作、手势、声音或词来解决问题，而不是只哭或抢。 | 问题解决；开关/打开/再来；分类和配对；户外探索 |
| 20个月 | 发展更稳定的词汇理解、简单选择、身体协调和自主生活能力。 | 二选一选择；身体部位和动作词；上下台阶保护练习；自主进食 |
| 21个月 | 把语言放进真实任务里：拿、放、给、找、帮忙、轮流。 | 生活任务语言；收纳和分类；小跑追球；睡前低兴奋流程 |
| 22个月 | 增强认知灵活性、假装游戏、社交等待和精细控制。 | 假装照顾娃娃；颜色/形状初体验；涂鸦和贴纸；等待和轮流 |
| 23个月 | 为2岁复盘做准备，巩固词汇、动作控制、自理和情绪表达。 | 2岁前复盘；双词组合萌芽；上下台阶和球类；告别奶瓶和少屏幕 |
| 24个月 | 做2岁阶段复盘，把会做的稳定下来，把担心的问题带去儿保。 | 2岁里程碑复盘；语言和听力评估意识；运动与自理检查；下一阶段计划 |
